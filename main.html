TIC-TAC-TOE

x plays first and o plays after x.

Click Anywhere In Compartment To Start.

Rules

  • 1. x plays first and o plays after x.
  • 2. Click on any place of your choice in 3*3 board to start the game.
  • 3. If same symbol- x / o- appears in line- vertically / horizantally / diagonally - x Wins / " Wins
  • 4. Refresh to start game again.
